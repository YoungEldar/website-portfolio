технологии которые использую

HTML5

Cтандартизированный язык разметки документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML.HTML — теговый язык разметки документов.

CSS3

Формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Преимущественно используется как средство описания, оформления внешнего вида веб-страниц.

JavaScript

Мультипарадигменный язык программирования. Наиболее широкое применение находит в браузерах как язык сценариев для придания интерактивности веб-страницам.

flex-box

Это модель веб-макета CSS3 .Гибкая компоновка позволяет автоматически настраивать адаптивные элементы внутри контейнера в зависимости от размера экрана (или устройства).

Grid

CSS-сетка - это методика в CSS, которая позволяет веб-разработчикам легче и последовательно создавать сложные адаптивные макеты веб-дизайна в браузерах.

Bootstrap

Свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации.

php7

Cкриптовый язык общего назначения, интенсивно применяемый для разработки веб-приложений. Является одним из лидеров среди языков, применяющихся для создания динамических веб-сайтов.

MySql

Это одна из самых популярных и самых распространенных СУБД (система управления базами данных) в интернете.Ее применение идеально для интернет сайтов, как небольших, так и достаточно крупных.

WordPress

Cистема управления содержимым сайта с открытым исходным кодом; сервер базы данных — MySQL; Сфера применения — от блогов до достаточно сложных новостных ресурсов и интернет-магазинов.
